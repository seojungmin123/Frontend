## 1. 관계(비교) 연산자 

= 는 value 할당

비교연산자의 반환값은 항상 boolean형임 (true or false)

== 보다는 === 쓰는게 좋음

연산자|설명|
---|---|
동등 (==)|피연산자가 서로 같으면 true를 반환합니다.
부등 (!=)|피연산자가 서로 다르면 true를 반환합니다.
일치 (===)|두 피연산자의 값과 타입이 모두 같은 경우 true를 반환합니다. Object.is와 JavaScript에서의 같음을 참고하세요.
불일치 (!==)|피연산자의 값 또는 타입이 서로 다를 경우 true를 반환합니다.
큼 (>)|왼쪽 피연산자가 오른쪽 피연산자보다 크면 true를 반환합니다.
크거나 같음(>=)|왼쪽 피연산자가 오른쪽 피연산자와 같거나 크면 true를 반환합니다.
작음 (<)|왼쪽 피연산자가 오른쪽 피연산자보다 작으면 true를 반환합니다.
작거나 같음 (<=)|왼쪽 피연산자가 오른쪽 피연산자와 같거나 작으면 true를 반환합니다.

## 2. 산술 연산자

*/이 +-보다 우선

**덧셈 연산자(+)** : 피연산자 중 하나라도 문자열이면, 숫자를 문자열로 변환한 후 두 문자열을 결합함.

예를 들어, `5 + "5"`는 `"5" + "5"`가 되어 결과는 `"55"`가 됨

<br>

**다른 산술 연산자(*, /, -)** : 숫자와 문자열이 함께 사용될 때, 문자열을 숫자로 변환하여 계산을 시도함.

 예를 들어, `10 * "5"`는 `10 * 5`가 되어 결과는 `50`이 됨

연산자|설명
---|---|
나머지 (%)|이항 연산자입니다. 두 피연산자를 나눴을 때의 나머지를 반환합니다.
증가 (++)|단항 연산자입니다. 피연산자에 1을 더합니다. 전위 연산자(++x)로 사용하면 피연산자에 1을 더한 값을 반환합니다.<br>후위 연산자(x++)로 사용한 경우 피연산자에 1을 더하기 전의 값을 반환합니다.
감소 (--)|단항 연산자입니다. 피연산자에서 1을 뺍니다. 반환 값은 증가 연산자처럼 동작합니다.
단항 부정 (-)|단항 연산자입니다. 피연산자의 부호를 반대로 바꾼 값을 반환합니다.
단항 플러스 (+)|단항 연산자입니다. 숫자 + 문자열이면 문자열 결합을 시도합니다.
거듭제곱 (**)|base^exponent, 즉 base를 exponent로 거듭제곱한 결과를 반환합니다.

## 3. 대입(복합, 할당) 연산자

+=, -=, *=, /=, %=
```js
let num = 10;

//num = num + 5;
num += 5; // 변수에 +5

console.log(num); // 15


let num = 10;

//num = num % 5;
num %= 5; // 변수를 5로 나눠 나머지를 구하겠다.

console.log(num); // 0
```

## 4. 논리 연산자

연산자|사용법|설명|
---|---|---|
논리 AND (&&)|expr1 && expr2|expr1을 false로 변환할 수 있으면 expr1을 반환합니다. 그 외의 경우에는 expr2를 반환합니다. 따라서 불리언 값과 함께 사용한 경우, 둘 다 참일 때 true를, 그 외에는 false를 반환합니다. 모든 값이 true 면 true. 즉, 하나라도 false 면 false를 반환함.
논리 OR (\|\|)|expr1 \|\| expr2|expr1을 true로 변환할 수 있으면 expr1을 반환합니다. 그 외의 경우에는 expr2를 반환합니다. 따라서 불리언 값과 함께 사용한 경우, 둘 중 하나가 참일 때 true를, 그 외에는 false를 반환합니다. 여러개 중 하나라도 true 면 true. 즉, 모든 값이 false 일때만 false를 반환함.
논리 NOT (!)|!expr|단일 피연산자를 true로 변환할 수 있으면 false를 반환합니다. 그 외에는 true를 반환합니다. true면 false, false면 true.

<br>

AND와 OR의 차이점 (평가)

- AND는 첫번째 false를 발견하는 즉시 평가를 멈춤.

- OR은 첫번째 true를 발견하는 즉시 평가를 멈춤.

<br>

우선순위

&&(AND)가 ||(OR)보다 우선순위가 높음. 먼저 실행.

코드의 가독성을 높이고 혼란 방지를 위해 항상 괄호()를 사용하여 그룹화 하는 것이 중요함!

**&&** : and **그리고**, **포함된 범위 설정**(x>=10 && x<=20)

**||** : or **또는**, **둘중에 하나의 범위**, **범위 초과 설정**
(x<0 || x>100) <- x가 0 이하이거나 100보다 클때

**!** : not **부정**
!true => false
!false => true
!(조건식-true/false) => false/true

```js
const gender = 'F';
const name = 'Jane';
const isAdult = true;

if (gender === 'M' && name === 'Mike' || isAdult) {
  console.log('통과');
} else {
  console.log('실패');
}
```

1. `gender === 'M' && name === 'Mike'`: 성별이 'M'이고 이름이 'Mike'인지를 체크함 > false
2. `&&` 보다 낮은 우선순위를 가지는 `||` 연산자가 나옴 > false || true
3. `isAdult`: 성인 여부를 체크함 > true '통과'

```js
// 이 조건식은 다음과 같이 그룹화 됨
(gender === 'M' && name === 'Mike') || isAdult
```

## typeof 연산자
```js
const name = "jungmin";
const list = ["mon","tue","wen"];
const myFun = new Function("5 + 2");

console.log(typeof 1); // "number"
console.log(typeof name); // "string"
console.log(typeof "wow"); // "string"
console.log(typeof true); // "boolean"
console.log(typeof undefined); // "undefinded"
console.log(typeof list); // "object"
console.log(typeof myFun); // "function"

// 키워드 null
// null은 객체(object)가 아님
// js 초기버전의 오류임. 하위버전의 호환성을 위해 냅둠 
console.log(typeof null); // "object"
```

## 6. 스프레드 연산자

spread 연산자(...)를 사용하면 기존 배열이나 객체의 전체 또는 일부를 다른 배열이나 객체로 빠르게 복사할 수 있음

```js
const fruitOne = ['apple', 'banana'];
const fruitTwo = ['grape', 'peach'];

// 기존 방법
const fruitAll = fruitOne.concat(fruitTwo);

console.log(fruitAll); // ['apple', 'banana', 'grape', 'peach']

// ES6 spread 연산자 활용 방법
const fruitAll = [...fruitOne, ...fruitTwo];

console.log(fruitAll); // ['apple', 'banana', 'grape', 'peach']
```

## 7. 삼항 연산자
삼항 연산자는 참/거짓에 따라 선택적으로 실행되는 조건문

`조건문 ? 선택문1 : 선택문2`

조건문이 참(True)이면 선택문1을 실행하고 조건문이 거짓(False)이면 선택문 2를 실행한다.

물음표를 여러개 사용하여 다중 삼항 연산으로 표현 가능

<br>

## 8. 우선순위

연산자|결합성|우선순위|
---|---|---|
++, --, !|오른쪽에서 왼쪽으로|높음
*, /, %|왼쪽에서 오른쪽으로|
+, -|왼쪽에서 오른쪽으로|
\> , >= , < , <=|왼쪽에서 오른쪽으로|
== , !=|왼쪽에서 오른쪽으로|
&&|왼쪽에서 오른쪽으로|
\|\||왼쪽에서 오른쪽으로|
= , += , -= , *= , /= , %=|오른쪽에서 왼쪽으로|낮음
