# 템플릿 리터럴 (Template literal)

내장된 표현식을 허용하는 문자열 리터럴

백틱을 사용 ``

<br>

## 1. 문자열삽입

- 백틱 사용X
```js
let = name = 'jungmin';
const greeting = 'Hello my name is' + name;
```

- 백틱 사용O
```js
let = name = 'jungmin';
const greeting = `Hello my name is ${name}`;
```

<br>

## 2. 표현식삽입

- 백틱 사용X
```js
let a = 1;
let b = 10;
console.log('1 * 10 is ' + (a * b));
```

- 백틱 사용O
```js
let a = 1;
let b = 10;
console.log(`1 * 10 is ${a * b}`);	
```

<br>

## 3. 여러 줄 문자열 생성

- 백틱 사용X
```js
let text = "Hello \
my name is jungmin \
how are you?";
```

- 백틱 사용O
```js
let text = `Hello
my name is jungmin
how are you?`;
```

<br>

## 4. 중첩 템플릿

```js

const people = [{
  name: 'jungmin 1',
  age: 17,
}, {
  name: 'jungmin 2',
  age: 18,
}, {
  name: 'jungmin 3',
  age: 19,
}
]; 

const markup = `
<ul>
	${people.map(person => `<li>  ${person.name}  </li>`)}
</ul>
`;

console.log(markup);

// <ul>
//   <li> jungmin 1 </li>,<li> jungmin 2 </li>, <li> jungmin 3 </li>
// </ul>

```

- map 함수를 이용하여 people의 각 원소에 대해 반복 동작을 수행, <br>
people 내의 name을 담아 li 표시

## 5. 삼항 연산자 추가

```js
// name, age와 함께 student 객체 생성
const student = {
  name: "jungmin",
  age: 18,
};

// student 객체에 song 프로퍼티가 있을 때만 문장에 추가하고, 
// 없으면 아무것도 반환하지 않는다.

const text = `
	<div>
	  <p> ${student.name} is ${student.age} years old ${student.song ?
              `and wrote song ${student.song}` : '' }
	  </p>
	</div>
`;

// <div>
//   <p> jungmin is 18 years old
//   </p>
// </div>
```

## 6. 템플릿 리터럴 내에 함수 전달

```js
const groceries = {
  meat: "pork chop",
  beggie: "salad",
  fruit: "apple",
  others: ['mushrooms', 'instant noodles', 'instant soup'],
};

// groceries의 각 값에 대해 map()을 수행하는 함수
function groceryList (others) {
  return `
	<p>
	  ${others.map( other => `<span>${other}</span>`).join('\n')}
	</p>
  `;
}

// p 태그 내 모든 groceries를 출력. 마지막은 others 배열의 모든 원소를 포함한다.

const markup = `
	<div>
	  <p>${groceries.meat}</p>
	  <p>${groceries.veggie}</p>
	  <p>${groceries.fruit}</p>
	  <p>${groceryList(groceries.others)}</p>
	</div>
`;

//  <div>
//	  <p>pork chop</p>
//	  <p>salad</p>
//	  <p>apple</p>
//	  <p>
//	     <p>
//		<span>mushrooms</span>
//		<span>instant noodles</span>
//		<span>instant soup</span>
//	     <p>
//	  <p>
//  </div>
```

## 7. 태그된 템플릿 리터럴

- 함수를 태그하여 템플릿 리터럴을 실행하면 템플릿 내부에 있는 모든 항목이 태그된 함수의 인수로 제공됨

- 템플릿 앞에 함수 이름을 쓰면됨

```js
let person = "jungmin";
let age = 18;

const myTag = (strings, personName, personAge) =>{
    let str = strings[1];
    let ageStr;

    personAge > 50 ? ageStr = "grandpa" : ageStr = "youngster";
    
    return `${person} ${str} ${ageStr}`;
}

console.log(myTag`That ${person} is a ${age}`); //jungmin  is a  youngster
```
