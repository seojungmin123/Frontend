## 1. 숫자 Number

정수와 실수 모두 Number 타입으로 표현함

- 사칙연산 가능
    
```js
console.log(1 + 2); // 더하기
console.log(5 - 2); // 빼기
console.log(2 * 2); // * 곱하기
console.log(6 / 2); // / 나누기
console.log(9 % 2); // % 나머지 값을 반환
```

## 2. 텍스트 String

“hello” < string  ( 'text' 와 "text"는 같음 )

- 항상 ' '나 " ", ` `로 감싸기
    
```js
const name1 = "Mike"; // 따옴표
const name2 = 'Mike'; // 작은따옴표
const name3 = `Mike`; // 백틱

const message = "I'm a boy"; // 안에 특수기호가 들어가는 경우 섞어 쓰기
const message2 = 'I'm a boy'; X
const message3 = 'I\'m a boy'; // 특수기호 앞에 \를 붙이면 인식함 (이스케이프)

// 문자열을 `백틱`으로 감싸면 표현식 삽입 가능
// ${변수이름 or 연산}
const name = "사과";
const price = 100;
const num = 3;

console.log(`${name}의 구매가는 ${price + num}원 입니다.`);
// ${}의 결과는 문자열로 자동 변환됨.
```

## 3. boolean (따옴표X)

true / false ( 참, 거짓 )

```js
console.log(Boolean('hi')); // 값이 존재하면 true
console.log(Boolean(true)); // true
console.log(Boolean(1234)); // true
console.log(Boolean(-1)); // true
console.log(Boolean(' ')); // 공백 있으면 true

console.log(Boolean('')); // 공백도 없으면 false
console.log(Boolean(0)); // false
console.log(Boolean(null)); // false
console.log(Boolean(false)); // false
console.log(Boolean(undefined)); // false
console.log(Boolean(NaN)); // false

console.log(Number(true)); // 1
console.log(Number(false)); // 0

console.log(parseInt(true)); // NaN
console.log(parseInt(false)); // NaN
```

## 6. NaN

not a number 숫자가 아님.

```js
// NaN인지 판별하기
const age = parseInt(prompt("How old are you?"));

console.log(isNaN(age)); // isNaN은 boolean을 반환함. NaN이면 true, 아니면 false.
```

+

**빈값선언은 null로.**
```js
let x = null; // 빈값선언
let y; // 값이 없는걸 꺼내올경우 undefined.
// undefined로 선언하지 않는다. 빈값 선언 null
console.log(y); // undefined
```
