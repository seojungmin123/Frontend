# 표기법
- 변수명, 클래스명 등 이름을 지을 때 사용하는 표기법
- 띄어쓰기가 불가하므로 단어를 알아볼 수 있도록 다양한 방식 사용

<br>

## 1. 카멜 표기법 (Camel Case)
- 단어의 첫 글자는 소문자로 시작하고, 그 뒤에 오는 단어의 첫 글자는 대문자로 시작
- 주로 변수명, 함수명 등 식별자에 사용

```
* camelCase
studentNumber
myVariableName
```

<br>

## 2. 파스칼 표기법 (Pascal Case)
- 모든 단어의 첫 글자가 대문자로 시작
- 주로 클래스명, 함수명 등에 사용

```
* PascalCase
StudentNumber
MyVariableName
```

<br>

## 3. 스네이크 표기법 (Snake Case)
- 모든 글자는 소문자로 작성하고, 단어사이에 `언더스코어`( _ )를 넣어 구분
- 주로 변수명, 함수명 등 식별자에 사용

```
* snake_case
student_number
my_variable_name
```

<br>

## 4. 케밥 표기법 (Kebab Case)
- 모든 글자는 소문자로 작성하고, 단어사이에 `하이픈`( - )를 넣어 구분
- 주로 html 속성명, URL 경로 등에 사용

```
* kebab-case
student-number
my-variable-name
```
