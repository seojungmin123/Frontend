## 가상이미지 (Placehold image)
- 실제 이미지 대신 작업중 빈공간을 채워줌
- 이미지가 로드 되지 않았을 때 사용자에게 이미지의 위치를 알려줌

<br>

### 1. placehold.co
기본적인 샘플 이미지

변경 가능한 속성 목록
- 크기
- 확장 포맷
- 배경과 텍스트 색상
- 텍스트 내용

사이트 url : https://placehold.co <br>
ex) : https://placehold.co/600x400

<br>

### 2. lorem picsum
무작위 샘플 이미지

변경 가능한 속성 목록
- 크기
- 특정 id 값 이미지
- 흑백(grayscale) 이미지
- 흐릿한(blur) 이미지
- 확장 포맷

사이트 url : https://picsum.photos/ <br>
ex) : https://picsum.photos/600/400
